# Birthday Roast 🎉

A fun and interactive web application that generates hilarious, personalized birthday roasts for your friends and loved ones!

## 🎯 Features

- **Personalized Roasts**: Generate custom, funny birthday messages based on user input
- **Interactive UI**: Clean and engaging user interface
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Dynamic Content**: Each roast is unique and tailored to the person
- **Share-Friendly**: Easy to copy and share roasts on social media

## 🛠️ Tech Stack

- **HTML5** - Semantic markup and structure
- **CSS3** - Styling, layout, and responsive design
- **JavaScript** - Dynamic functionality and roast generation logic

## 📋 Prerequisites

- Any modern web browser (Chrome, Firefox, Safari, Edge)
- No installation required!

## 🚀 How to Use

1. **Clone or Download** this repository:
   ```bash
   git clone https://github.com/duakhurram2212/birthday-roast.git
   ```

2. **Navigate** to the project folder:
   ```bash
   cd birthday-roast
   ```

3. **Open** the `index.html` file in your web browser:
   - Double-click `index.html`, or
   - Right-click → Open with → Your preferred browser

4. **Generate Roasts**:
   - Enter your friend's name or details
   - Click the "Generate Roast" button
   - Copy the roast and share with your friends!

## 💡 How It Works

1. User inputs a name or description
2. JavaScript generates a random, funny roast from a pool of templates
3. The roast is displayed on the screen
4. User can copy and share the roast

## 🎨 Features Breakdown

- **Random Roast Generator**: Uses JavaScript to pick random roasts
- **Copy to Clipboard**: Easy one-click copy functionality
- **Mobile Friendly**: Touch-optimized interface
